# 20230117-javascript中的数据类型到底有几种？

直接给出答案：截止至2023年，一共**8种**数据类型。

1. `Number`  
2. `String`
3. `Boolean`
4. `undefined`
5. `object`
6. `Null`
7. `Symbol`  
8. `bigInt`
