# 折线图

## 效果一 ：折现+渐变区域：

![image-20250814142554777](https://s2.loli.net/2025/08/14/Ahx1ViBdzoJGy4q.png)

​	实现要点：

- 图表大小配置

  ```js
   // 网格配置
      grid: {
        left: 0,
        right: 0,
        top: 10,
        bottom: 50,
        containLabel: true,
      },
  ```

- 背景色透明

  ```js
  {
     // 背景透明
      backgroundColor: 'transparent',
  }
  ```

- X轴

  ```js
  // X轴配置 - 时间轴
      xAxis: {
        type: 'category',
        data: mockData.map((item) => item.time),
        axisLine: {
          show: false, // 隐藏x轴的横线
        },
        axisTick: {
          show: false, // 隐藏刻度线
        },
        axisLabel: {
          color: '#FFFFFF',
          fontSize: 12,
          fontFamily: 'D-DIN-PRO',
          fontWeight: 'bold',
          margin: 8,
        },
      },
  ```

  

- Y轴

  ```js
  // Y轴配置 - 风速数值
      yAxis: {
        type: 'value',
        min: 0,
        max: 100,
        interval: 20,
        axisLine: {
          show: false,
        },
        axisTick: {
          show: false,
        },
        axisLabel: {
          color: '#FFFFFF',
          fontSize: 12,
          fontFamily: 'D-DIN-PRO',
          fontWeight: 'bold',
          margin: 8,
          formatter: '{value}',
        },
        // 分隔线样式 - 虚线网格
        splitLine: {
          show: true, // 分隔线显示
          lineStyle: {
            color: 'rgba(255, 255, 255, 0.2)',
            type: 'dashed', // 虚线
            width: 1,
          },
        },
      },
  ```

- 线条加点

  点的样式设置：
  https://echarts.apache.org/zh/option.html#series-line.symbol

  ECharts 提供的标记类型包括

  ```
  'circle'`, `'rect'`, `'roundRect'`, `'triangle'`, `'diamond'`, `'pin'`, `'arrow'`, `'none'
  ```

  设置:`symbol:'none'`时，不显示点
  ![image-20250814145450803](https://s2.loli.net/2025/08/14/QFVzNpTxqSI2rjn.png)

- 折线图的高亮状态

  > https://echarts.apache.org/zh/option.html#series-line.emphasis

  ```js
   // 高亮状态
  emphasis: {
    itemStyle: {
      shadowBlur: 10,
      shadowColor: 'rgba(48, 226, 232, 0.6)',
      shadowOffsetX: 0,
      shadowOffsetY: 4,
    },
  },
  ```

  

- 线条下的区域渐变

  不加则不显示

  ```js
  // 区域渐变
      areaStyle: {
        color: {
          type: 'linear',
          x: 0,
          y: 0,
          x2: 0,
          y2: 1,
          colorStops: [
            {
              offset: 0,
              color: 'rgba(48, 226, 232, 0.3)',
            },
            {
              offset: 1,
              color: 'rgba(48, 226, 232, 0.05)',
            },
          ],
        },
      },
  ```
