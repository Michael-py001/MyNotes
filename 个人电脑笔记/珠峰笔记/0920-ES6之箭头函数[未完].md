## 箭头函数

```js
let 函数名 = function (形参1, 形参2, ...) {};
let 函数名 = (形参1, 形参2, ...) => {};
function fn(x) {
    return x + 10;
}
let fn = (x) => {
    return x + 10;
};
let fn = x => {
    // 只有一个形参可以省略小括号
    return x + 10;
};
如果函数体中只有一句话而且是return xxx，可以简写为下面方式
let fn = x => x + 10;
fn(10);

function fn(x) {
    return function (y) {
        return x + y;
    }
}
const fn = x => {
    return y => {
        return x + y;
    };
};
const fn = x => {
    return y => x + y;
};
const fn = x => y => x + y;
```
