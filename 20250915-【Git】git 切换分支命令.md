# 20250915-【Git】git 切换分支命令

切换 Git 分支通常使用 `git checkout` 或 `git switch` 命令。

## **1. 切换到已存在的分支:**

你可以使用 `git checkout` 命令，后面跟上你要切换到的分支名。

```bash
git checkout <分支名>
```

例如，切换到名为 `develop` 的分支：

```bash
git checkout develop
```

或者，使用较新的 `git switch` 命令，这个命令的用途更专一，就是用来切换分支：

```bash
git switch <分支名>
```

例如：

```bash
git switch develop
```

## **2. 创建新分支并立即切换过去:**

如果你想创建一个新的分支，并马上切换到这个新分支上，可以使用带 `-b` 参数的 `git checkout` 命令。

```bash
git checkout -b <新分支名>
```

例如，创建名为 `feature/new-login` 的新分支并切换过去：

```bash
git checkout -b feature/new-login
```

同样，你也可以使用 `git switch` 配合 `-c` (create) 参数达到同样的效果：

```bash
git switch -c <新分支名>
```

例如：

```bash
git switch -c feature/new-login
```

## **总结:**

*   **`git checkout <branch>`**: 切换到现有分支。
*   **`git checkout -b <branch>`**: 创建并切换到新分支。
*   **`git switch <branch>`**: (推荐) 切换到现有分支。
*   **`git switch -c <branch>`**: (推荐) 创建并切换到新分支。

如果你不确定当前有哪些分支，可以运行 `git branch` 或 `git branch -a` 来查看本地和所有远程分支。
