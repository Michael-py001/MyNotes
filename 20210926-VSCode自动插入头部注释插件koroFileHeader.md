# 20210926-VSCode自动插入头部注释插件koroFileHeader

## 安装插件 ：koroFileHeader

## 配置

设置中心搜索 `korofileheader`，或者在插件页面进入设置

```js
 "fileheader.customMade": {
        "Author": "Wushili", 
        "Date": "",//自动生成
        "LastEditors": "Wushili",
        "LastEditTime": "",//自动生成
        "Description": "",
        "FilePath": ""//自动生成
    },
```

```json
"fileheader.configObj":{
	"autoAdd":true  //是否自动添加
}
```

## 快捷键

windows:

```
window`：`ctrl+alt+t`,
`mac`：`ctrl+cmd+t`,
`linux`: `ctrl+meta+t
```

`ctrl + alt + i` : 插入头部注释

`ctrl + alt + t` : 添加函数注释
